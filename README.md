# Civilian DLI Method

A research project to investigate, reverse-engineer, and adapt the language-training methods associated with the U.S. Defense Language Institute Foreign Language Center (DLIFLC) for civilian self-study, with a later Japanese-specific implementation.

## Project goals

1. Reconstruct how DLI language training actually works using credible evidence.
2. Separate verified facts from anecdotes, inference, and internet folklore.
3. Identify the instructional mechanics that can be reproduced outside a military institution.
4. Design a realistic civilian study system.
5. Adapt the resulting framework for Japanese and AI-assisted language learning.

## Evidence standard

Every important claim should be assigned one of these statuses:

- **Verified** — directly supported by a primary or authoritative source.
- **Strongly supported** — supported by several credible, independent sources.
- **Anecdotal** — reported by students, graduates, or instructors but not independently established.
- **Inferred** — a reasoned conclusion drawn from available evidence.
- **Unverified** — plausible but currently unsupported.
